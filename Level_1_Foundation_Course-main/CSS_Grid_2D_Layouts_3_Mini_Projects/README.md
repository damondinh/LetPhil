# CSS Grid to Design (2‑D Layouts) — 3 Mini Projects

## Topics covered (matches your screenshot)

- Enabling CSS Grid (`display: grid`)
- Defining columns and rows (`grid-template-columns`, `grid-template-rows`)
- Placing items (`grid-column`, `grid-row`)
- Aligning items (horizontal + vertical alignment)
- Advanced layouts with `grid-template-areas`

---

## How to use

1. Open `Project_01.../MENTEE/index.html` in your browser
2. Follow the steps inside `MENTEE/styles.css`
3. Save → refresh after each step

---

# Project 1 — Grid Basics Gallery (Columns + Rows)

You’ll build a simple 2‑D “tile gallery” using:

- `display: grid`
- `grid-template-columns`
- `grid-template-rows`
- `gap`

---

# Project 2 — Grid Placement Lab (grid-column / grid-row + alignment)

You’ll build a layout where items “span” across rows/columns:

- `grid-column: start / end`
- `grid-row: start / end`
- `justify-items` / `align-items`
- `justify-content` / `align-content` (optional but super helpful)

---

# Project 3 — Dashboard Layout (grid-template-areas)

You’ll build a real dashboard wireframe using:

- `grid-template-areas`
- named areas like `header`, `sidebar`, `main`, `panel`, `footer`
- responsive switch to stacked layout with a simple media query

---

## Tiny glossary (super beginner-friendly)

- **Grid container**: the parent with `display: grid`
- **Grid items**: the children inside the grid container
- **Tracks**: columns/rows you define in the grid
- **Gap**: space between grid items
- **Spanning**: making an item cover multiple columns/rows
